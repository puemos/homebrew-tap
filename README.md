# Puemos Tap

## Available Formulae

| Formula | Description |
| --- | --- |
| [lareview](Formula/lareview.rb) | Desktop code review companion with AI-generated plans |

## How do I install these formulae?

`brew install puemos/tap/lareview`

Or `brew tap puemos/tap` and then `brew install lareview`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "puemos/tap"
brew "lareview"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
