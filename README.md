# Puemos Tap

## Available Formulae

| Formula | Description |
| --- | --- |
| [lareview](Formula/lareview.rb) | Desktop code review companion with AI-generated plans |
| [sshoosh](Formula/sshoosh.rb) | Self-hosted SSH/TUI workspace chat |

## How do I install these formulae?

`brew install puemos/tap/lareview`

`brew install puemos/tap/sshoosh`

Or `brew tap puemos/tap` and then `brew install lareview` or `brew install sshoosh`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "puemos/tap"
brew "lareview"
brew "sshoosh"
```
