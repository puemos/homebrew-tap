# Puemos Tap

## How do I install these formulae?

`brew install puemos/tap/<formula>`

Or `brew tap puemos/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "puemos/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
